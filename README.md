
# Tanks

Десктопная игра танчики.

## Запуск приложения

- Скачайте и запустите этот файл.
- Если вы клонировали проект, то файл для запуска находится по пути out\artifacts\TanksOnline.jar


## Управление 
- движение - стрелки вверх, вниз, вправо, влево.
- выстрел - пробел
- воскреситься - Num0
